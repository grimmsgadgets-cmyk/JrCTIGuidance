# Junior CTI Guide

This repository contains a GitHub Pages-ready static website for the junior CTI analyst guide, "Baselining and Tracing Adversary Behavior." It is a field guide / analyst handbook for turning reporting into actor baselines, change assessments, and stakeholder action.

## File Structure

```text
.
├── index.html
├── styles.css
├── script.js
├── .nojekyll
└── README.md
```

## Preview Locally

Open `index.html` directly in a browser.

This is a static site and does not need a backend, paid service, framework, dependency install, or build step.

## Publish With GitHub Pages

1. Push files to GitHub.
2. Go to repository Settings.
3. Go to Pages.
4. Select Deploy from branch.
5. Select main branch and root folder.
6. Save.

GitHub Pages will serve the site from the repository root.
